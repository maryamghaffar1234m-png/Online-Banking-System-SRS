# Online-Banking-System-SRS
Software Requirements Specification for Online Banking System

## 📌 Project Overview

This project contains the Software Requirements Specification (SRS) for an Online Banking System. It defines system functionalities, user interactions, and constraints.

## 🎯 Features

* User Authentication (Login System)
* Account Management
* Fund Transfer
* Transaction History
* Security & Performance Requirements

## 👥 Users

* Customers
* Admin
* Bank Staff
